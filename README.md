# Movie Recommendation System using Graph Neural Networks

## Overview

This project develops a personalized movie recommendation system using Graph Neural Networks (GraphSAGE) and PyTorch Geometric. The model learns user preferences from historical ratings and predicts future movie ratings based on user-movie interactions.

## Dataset

- MovieLens Latest Small Dataset
- User ratings and movie metadata
- Movie genres and movie titles

## Technologies

- Python
- PyTorch
- PyTorch Geometric
- GraphSAGE
- Sentence Transformers
- Pandas
- NumPy

## Features

- Graph-based recommendation system
- Heterogeneous graph construction
- Movie title embeddings using Sentence Transformers
- Genre feature encoding
- User-movie interaction modeling
- Rating prediction

## Evaluation

- Train / Validation / Test split
- RMSE evaluation metric
